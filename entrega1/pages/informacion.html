Notación BEM

Todo el código CSS y HTML se escribió utilizando la notación BEM (Block - Element - Modifier). Esta convencion resulta una manera conveniente de organizar el código, mejorando su legíbilidad, facilitando la cooperacion entre desarrolladores y generando un código más consistente. BEM establece que son las clases, y no los IDs, las que deben ser usadas para referenciar los elementos html en el código CSS, adicionalmente se debe adoptar una manera particular de construir clases, teniendo en cuenta el tipo de elemento al que se van a aplicar y el significado que las propiedades definidas, en su conjunto, tienen respecto al elemento:

  • Block

    Un bloque corresponde al contenedor o contexto en el que un elemento se encuentra. Corresponde a las unidades estructurales más grandes del layout de la página. Se denota con el nombre de la clase, sin sufijos. 
    .block

  • Element

    Un elemento es una pieza de bloque. Se deora
    Se denota con el nombre del bloque seguido de 2 barras bajas y el nombre del elemento. 
    .block__element

  • Modifier

    Un modificador corresponde a las variaciones que puede tener un mismo elemento. Se denota con la clase del elemento seguida de 2 guiones y el nombre de la variacion. 
    .block__element--modifier
